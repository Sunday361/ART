# Graduation Project

## This is a project of In-Memory Database.

### 1. Use ART Tree as Index.
### 2. Use DAF as Transaction framework.
### 3. Use PAX as BLOCK Layout

All Data is stored in memory, but I want to add write ahead log for recovering data

Art-Index is used as secondary-index, the primary index should be B-tree or Bw-tree
